# MECCHA CHAMELEON Fan Guide Site

This is a static fan-guide concept inspired by a wiki/tips page and a Steam store page. It was prepared for EZWebStudioShop as a polished MECCHA CHAMELEON strategy hub.

## Open Locally

Open `index.html` in a browser. No build step is required.

## Included Sections

- Game bio and Steam-style tags
- Beginner tips
- Hide and seeker strategy categories
- Official map guidance
- Player review summary
- Estimated global reach map
- Screenshot gallery

## Important Note

This concept uses public Steam media URLs and public Steam data. For a long-term site, replace external media links with your own licensed/local assets where possible.
